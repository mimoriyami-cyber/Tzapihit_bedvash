# Camp Registration & Management System 🏕️

A full-stack web application designed to streamline the registration process and administrative management for a summer camp.

## 🚀 Features
* **User Registration:** Simple and intuitive registration form for campers.
* **Admin Dashboard:** Secure area for administrators to view, edit, and manage registrations.
* **Real-time Database:** Powered by Supabase for reliable and fast data storage.
* **Security:** Sensitive configuration managed via environment variables (protected).

## 🛠️ Tech Stack
* **Frontend:** React.js, CSS3
* **Backend:** Node.js, Express
* **Database:** Supabase (PostgreSQL)
* **Version Control:** Git & GitHub

## 📁 Project Structure
* `/CampSystem/client`: React frontend application.
* `/camp-server`: Node.js backend server and API logic.
